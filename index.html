Example of Tabs with Automatic Activation

This Github repository Tabs is a forked and adjusted example of tabs with automatice activation from WAI ARIA Practices by W3.org. I've added the ability to direct linking to a specific tab. The original files are located on the Github repo aria-practices of W3c.
